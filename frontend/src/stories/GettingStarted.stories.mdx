### Welcome to Blade's Storybook
# Frontend Development for LedFx

This is a growing place for supporting the frontend development of LedFx

## Requirements

- Running LedFx Core
- Node >= 14

## Getting started

`yarn` is recommended, but you can use `npm` aswell
```bash
git clone https://github.com/YeonV/LedFx-Frontend-v2
cd LedFx-Frontend-v2
yarn # npm install
yarn start # npm run start
```

When working with VsCode, you can make use of tasks:
```bash
git clone https://github.com/YeonV/LedFx-Frontend-v2
cd LedFx-Frontend-v2
code .
```
`CTRL`+`SHIFT`+`P` -> Run Task<br />
-> Init Workspace (this does the yarn install)<br />
-> Start Frontend (yarn start)
